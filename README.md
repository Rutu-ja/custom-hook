# Custom Hooks Project

This project demonstrates the use of custom React hooks to manage state and side effects in a React application.

## Folder Structure

The project has the following folder structure:

```
custom-hooks-2/
├── public/
│   ├── index.html
│   └── ...
├── src/
│   ├── hooks/

│   │   ├── useCustomHook2.js
│   │   └── ...
│   ├── components/
│   │   ├── Component1.js
│   │   ├── Component2.js
│   │   └── ...
│   ├── App.js
│   ├── index.js
│   └── ...
├── package.json
└── README.md
```
